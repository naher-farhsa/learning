GenAI: Fields of AI models that can create new content - such as text, images, audio, video or code in natural language (human-like data).

Technologies:
- LLM Apps: ChatGPT
- Diffusion models (Image): MidJourney
- Codi models: CodeLLama
- Text-to-Speech models: Eleven Labs v3
- Video gen models: Sora

Traditional AI: About finding patterns in data and giving predictions.                  

Generative AI: About learning distribution of data so that it can generate a new sample from it with human like response. 

Agentic AI: About creating system that use LLM as "reasoning engine" and tools as "action items" to achieve specific goals. It can reason, plan and take action.

Characteristics of Agentic AI:
- Autonomy:  Execution, Decision making(HITL), Tool Usage
- Goal Oriented: Objective into Action with constraint
- Planning: Break down high-level goal into structured sequence of actions
- Reasoning: Cognitive process to interpret info, make decision, conclusion 
- Adaptability: Modify plans, strategy and action for unexpected condition
- Context Awareness: Understand, retain and utilize relevant info(STM, LTM)

Components of Agentic AI:
1) Brain (LLM): 
   - Goal Interpretation
   - Planning
   - Reasoning
   - Tool Selection
   - Communication

2) Orchestrator (Langgraph): 
   - Task Sequencing
   - Conditional routing
   - Retry logic
   - Looping
   - Delegation

3) Tools:
   - External Actions
   - Knowledge Base Access

4) Memory: 
   - Short Term Memory
   - Long Term Memory
   - State Tracking

5) Supervisor:
   - Approval Requests (HITL)
   - Guardrails Enforcement
   - Edge Case Escalation
