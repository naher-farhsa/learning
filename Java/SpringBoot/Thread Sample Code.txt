
A thread is a small worker inside a process that executes tasks. One process can have many threads working together.

Single Thread: Single task + Unique execution flow
Multi Thread : Multiple task + Independent execution flow 

NEW -> RUNNABLE |->RUNNING->TERMINATED
              ^-|-> BLOCKED/WAITING

Synchronizaton : lock resource to one thread per use

volatile : visible but not atomic to r/w
synchronized : visible + atomic

import java.util.*;
class Hello implements Runnable{
    public void run(){
       for(int i=0;i<1000000;i++){
        System.out.println(i+"Hello");
        }
    }
    
}
class World implements Runnable{
    public void run(){
        for(int i=0;i<1000000;i++){
        System.out.println(i+"World");
        }
    }
    
}
class Main {
    public static void main(String[] args) {
        System.out.println("Try programiz.pro");
        Hello h=new Hello();
        Thread t1=new Thread(h);
        World w=new World();
        Thread t2=new Thread(w);
        t1.start();
        t2.start();
        for(int i=0;i<1000;i++){
            System.out.println(i+"Main");
        }
        
    }
}
